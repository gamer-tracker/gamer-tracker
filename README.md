## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação das páginas (página inicial, navegação, formulários).
* **CSS3:** Estilização externa, layout responsivo, cores e tipografia.
* **JavaScript:** Validação de formulários, máscaras, contadores, cálculos simples e manipulação de DOM.
* **PHP:** Processamento de formulários via requisições POST e operações de back-end.
* **Banco de Dados:** ultilizado o MySQL.
* **API Consumida:** **RAWG Video Games Database API** - Utilizada para consultar dados como ano de lançamento e gênero dos jogos no sistema.

## ✨ Funcionalidades
Conforme os requisitos da disciplina, o sistema contempla:
* **Cadastro:** Inserção de novos dados através de formulários.
* **Listagem:** Visualização estruturada dos jogos cadastrados.
* **Edição/Exclusão:** Simulação ou execução real das ações no sistema.
* **Integração com API:** Consulta de dados externos (RAWG API) para complementar a aplicação.

## ⚙️ Instruções para Execução

### Pré-requisitos
Para rodar este projeto localmente, é necessário possuir um servidor web local que suporte PHP (como [XAMPP](https://www.apachefriends.org/pt_br/index.html), [WampServer](https://www.wampserver.com/en/) ou [Laragon](https://laragon.org/)).

### Passos para rodar localmente
1. Clone este repositório no seu computador:
   ```bash
   git clone https://github.com/gamer-tracker/gamer-tracker.git
   ```
2. Mova a pasta contendo os arquivos do projeto para o diretório raiz do seu servidor web (ex: `htdocs` no XAMPP ou `www` no WampServer).
3. Inicie os serviços do servidor web (Apache e MySQL, se estiver usando banco de dados).
4. Abra o seu navegador de preferência e acesse o endereço correspondente:
   ```text
   http://localhost/gamer-tracker
   ```
